# GitHub 仓库创建指南（小白版）

## 步骤一：登录GitHub
1. 打开浏览器，访问：https://github.com
2. 点击右上角 **Sign in** 登录您的账号

## 步骤二：创建新仓库
1. 登录后，点击右上角的 **+** 号
2. 选择 **New repository**（新建仓库）

## 步骤三：填写仓库信息
- **Repository name**（仓库名称）：输入 `copilot-desktop`
- **Description**（描述）：可填写 `Microsoft Copilot 桌面应用`
- **Public/Private**：选择 **Public**（公开）- 免费账号必须选择公开才能使用GitHub Actions
- **重要**：❌ **不要勾选** "Add a README file"
- **重要**：❌ **不要勾选** "Add .gitignore"
- **重要**：❌ **不要勾选** "Choose a license"

## 步骤四：创建完成
点击 **Create repository** 按钮

## 创建完成后
您会看到一个页面，上面有一个类似这样的地址：
```
https://github.com/您的用户名/copilot-desktop.git
```

**请将这个地址告诉我，格式如下：**
```
我的GitHub用户名是：xxx
我的仓库地址是：https://github.com/xxx/copilot-desktop.git
```

我会帮您完成剩下的所有操作！
